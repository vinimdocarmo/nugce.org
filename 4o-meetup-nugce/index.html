Talks

Iniciando uma aplicação web com React, Webpack e Babel
Veja os componentes da sua aplicação serem atualizados em tempo real. Use todas as novas features do ES6 e entenda como uma aplicação pode ser totalmente quebrada em componentes e funcionar de forma totalmente independente.
Por Vitor Queiroz

Acesso não bloqueante com node.js: Teste de performance entre um ambiente orientado a eventos e um ambiente multithread
Este tema foi minha defesa de monografia, onde foi aplicado um teste entre duas aplicações, ambas com o mesmo proposito, onde uma dessas aplicações foi escrita em node.js e a outra em Java, utilizando servlet.
Por Romário Teles

Programação Orientada a Aspecto com FeathersJS
Crie uma API REST tão rapido quanto um Parse ou Firebase sem deixar de ser DRY quando as regras de negocio chegarem.
Conheça o framework FeatherJS e como a programação orientada a aspecto pode ajudar seu codigo a crescer de forma sustentavel.
Por Abraão Alves

Node.js & Google Cloud Platform
Nesta talk serão abordadas as configurações mínimas para se fazer deploy de aplicações Nodejs em ambente Google, bem como algumas boas práticas pra isso.
Por Cícero Viana

BrasilJS Conf 2016 - Overview
Saiba o que rolou na BrasilJS Conf 2016
Por CJ Patoilo, Ian Ramos e uma cambada de outros devs cearenses que marcaram presença na conferência.
